# Echo Craft Form System

Version: 1.0

---

## Purpose

Provides reusable form controls for all Echo Craft Framework websites.

The Form System ensures every contact form, quote request, booking form,
and client inquiry maintains a consistent premium appearance.

• Labels

• Inputs

• Textareas

• Select Menus

• Checkboxes

• Radio Buttons

• Submit Buttons

• Validation States
