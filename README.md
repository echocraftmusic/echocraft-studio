# EC Framework v1.0

## Purpose

EC Framework is the internal website foundation used to build all Echo Craft and client websites.

## Core Structure

- index.html
- about.html
- services.html
- portfolio.html
- contact.html
- blank.html
- 404.html

## JavaScript

- main.js
- navigation.js

## Core Principles

- One responsibility per file.
- Reuse before rebuilding.
- Mobile-first.
- Performance first.
- Premium design.
- Easy maintenance.

## Workflow

1. Copy EC Framework.
2. Rename project.
3. Replace branding.
4. Replace content.
5. Launch.
